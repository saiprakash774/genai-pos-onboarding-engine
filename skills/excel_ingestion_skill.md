# Excel Ingestion Skill

## Procedural Instructions

When executing this skill to extract and normalize menu data, the agent MUST take the following steps:

- **Step 1: Ingest**
  Ingest the target `.xlsx` file using `pandas`.

- **Step 2: Handle Merged/Empty Cells**
  Strictly use the `ffill()` (forward-fill) method on the "Base_Drink" and "Category" columns so that subsequent rows with size variations correctly inherit their parent data.

- **Step 3: Map Modifiers**
  Isolate the "Modifiers" data block and enforce the 'Applies_To' relational mapping logic, ensuring add-ons are strictly tied to permitted base categories.

- **Step 4: Output Validation**
  The script must safely drop completely empty rows (all NaN) and hold the data in memory for the Validation Agent.

**CRITICAL REQUIREMENT:**
The code generated by this skill MUST pass the Gherkin rules defined in `/specs/menu_mapping.feature`.
